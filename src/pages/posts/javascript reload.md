# JAVASCRIPT 快速在控制台定时刷新和执行网页任务 简单快捷的命令
这段代码可以在浏览器控制台定时循环的执行一段代码和动作
```javascript
let intervalId = setInterval(function(){ 
    location.reload(); 
}, 10000);

// 当你想要停止刷新时，运行以下代码：
clearInterval(intervalId);
```
